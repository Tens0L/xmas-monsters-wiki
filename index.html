01 / Introduction

はじめに

物語のはじまりと、この手帖の読みかた。

物語は、クリスマスイブの夜からはじまります。主人公はサンタさんに 「おもちゃはいらないので、勇者になって世界中を旅する冒険映画が見たい」という てがみを書いて眠りにつき、目が覚めると見知らぬ森の中に倒れていました。 目の前では、小さなスライムが悪いモンスターと戦っている――そこから、 すべてのモンスター集めとぼうけんがはじまります。

本作はビルド不要・外部素材ゼロで動くブラウザRPGで、グラフィックは16×16ドットの タイルとスプライトをすべてコードで生成しています。この手帖に貼ってあるモンスターの アイコンも、実際のゲームと同じ生成ロジックでその場で描いたものです。

表示言語は English / にほんご をタイトル画面や「せってい」から いつでも切り替えられます（デフォルトは英語）。この手帖ではモンスターと技の名前を 日本語名・英語名の両方で載せているので、どちらの言語で遊んでいても迷わないはずです。

02 / Controls

そうさ方法

必要なキーはこれだけ。フィールドでもバトルでも共通です。

↑↓←→ / WASDマス目移動・カーソル
Spaceけってい／ミニメニューを開く
X / ESCキャンセル

フィールドで正面に人・看板・カウンターなど「反応できるもの」があるとき、Spaceキーは そのままミニメニューを飛ばして直接話しかけます。何もない場所でSpaceを押すと、 はなす/しらべる・モンスター・どうぐ・せってい・とじるのミニメニューが開きます。

「モンスター」メニューの最後にあるじんけいでは、バトル開始時の 初期配置をバトル画面と同じ10×10マスの左3列の中で自由に組めます。前衛・後衛を 自分で決められるので、打たれ弱い回復役は後ろに置くと安定します。

03 / Walkthrough

ぼうけんの流れ

オープニングからクリア後コンテンツまでの一本道。

森でスライムと出会う

目覚めた森で、悪いモンスターと戦うスライムに手を貸すとチュートリアルバトルが始まります。 バトル画面の使い方を吹き出しで説明してくれるので、その通りに戦えば勝てます。勝利後、スライムが最初の仲間になります。

カステラ城下町に到着

森を北へ抜けると城下町。やどや・どうぐや・まほうつかいの家・ぼくじょうが並び、 ここが以後の拠点になります（詳しくは次の章）。

お城で王様に話す

城の王様に会うと、城の裏にあるどうくつにあふれるモンスターの討伐を頼まれます。 話し終えると、みはりの兵士が退き、どうくつへ入れるようになります。

どうくつ B1・B2を進む

ランダムエンカウントの2階層。入口付近は弱い敵が1体だけですが、 奥へ進むほど同時に出る数（最大3体）もレベルも上がっていきます。焦らず入口周辺で レベルを整えてから奥を目指すのがコツです。

最深部でブラックサンタ戦

どうくつの主、ブラックサンタとその手下との戦い。 よい子の夢を奪う存在として立ちはだかります。属性の弱点を突けるパーティで挑みましょう。

王様からのほうび

討伐後、王様の前にワープしてお礼を言われ、伝説のモンスター ホーリーナイト（Lv12）が仲間になります。

クリア後 POSTGAME

改心したブラックサンタが最深部に居座っており、再戦を挑んで勝つと今度こそ仲間にできます。 さらにクリア後限定で、こんとんのりゅうとイヴという2体の 真の最終合成モンスターが解禁されます（09章）。

街の施設

カステラ城下町でできること。ぼうけんの合間は必ずここに戻ってこよう。

🛏 やどや

「たびのきろく」でセーブ、「ひとやすみ」（20G）でパーティ全員のHP・MP・状態異常を全回復。 メニューは上からひとやすみ → たびのきろく → やめるの順です。

🎒 どうぐや

アイテムの売買。「かう」画面ではカーソルを合わせた品の説明が下に表示されます。

  • やくそう（HP+30）30G
  • どくけしそう（どく治療）10G
  • まほうのみず（MP+20）80G
  • かえりのはね（城下町へワープ）25G
  • ちからのたね（こうげき+3）300G

🧪 まほうつかいの家

Lv10以上のモンスター2ひきを「まほうのなべ」に入れると、 新しいモンスターが生まれます（09章で詳しく）。ごうせい表もこの場で確認できます。

🐑 ぼくじょう

手持ちが3ぴきいっぱいのときに預けておける場所。あずかり中のモンスターは 話しかけると右側にリスト表示され、いつでも入れ替えできます（最後の1ぴきは預けられません）。

05 / Battle System

バトルシステム

10×10マスの自動タクティクスバトル。指示するのは「さくせん」だけ。

エンカウントすると、10×10マスのフィールドに味方は左側・敵は右側で対峙します。 行動順はすばやさ（AGI）の高い順。各モンスターに移動・攻撃・とくぎの 指示を出すのではなく、次の3つの「さくせん」を割り当てて、あとは自動で戦ってもらいます。

ガンガンいこうGo all out

最もダメージの出る手段（とくぎ優先度が高い）を選び、4割の確率でHPが一番低い敵を狙う。とにかく速攻したいときに。

いろいろやろうMix it up

HP40%未満の味方がいれば回復を優先。3割強でどく技、5割で攻撃とくぎを織り交ぜる、バランス型。

いのちをだいじにStay safe

HP60%未満の味方をまず回復。自分のHPが25%を切ると距離を取る、堅実型。回復役に向く。

バトル中にSpaceを押すと、次のラウンドから全員のさくせんを変更できます （画面下にも操作説明が常時表示されます）。攻撃時には属性ごとに専用パーティクルが出るほか、 バトルの演出速度は「せってい」のバトルスピード（ゆっくり／ふつう／はやい）で 調整可能です。デフォルトはゆっくりめの「ふつう」。

状態異常もいくつかあります。どくは毎ラウンド最大HPの8%を削り、 しばり（ルートバインド等）は確率で1ターン行動不能に、 アイアンウォールは自分の防御を3ラウンドの間1.5倍にします。

06 / Elements

ぞくせい相性

火・水・木・金・土・光・闇の7属性。攻撃側と防御側の相性でダメージが変わります。

五行の相剋にならい、火は金を、金は木を、木は土を、土は水を、水は火をそれぞれ打ち破ります。 加えて光と闇はおたがいに強く効きます。得意な属性で攻めるとダメージ1.5倍、 不利な属性を受けると0.65倍です。

10 / Strategy

せんりゃくのコツ

遊びながら見えてきた、覚えておくと得するポイント。

01
同じ種族を2ひき、Lv10まで育てるだけで確実に上位種へ進化できます。合成は運試しではなく、 どの系統を伸ばすか決める育成プランです。ねらいの属性を1本決めてから野生で仲間を集めましょう。
02
とくぎの消費MPは全体的に低め（基本技は2〜5MP）なので、序盤から遠慮なく撃って大丈夫です。 MPが尽きたときの保険に「まほうのみず」を1〜2個持っておくと安心。
03
「いろいろやろう」の回復役はじんけいで後列（右寄りの列）に置くと、 敵の初撃を受けにくくなります。前衛はDEFの高いゴーレム系・タイタン系がおすすめ。
04
どうくつは入口付近で足踏みしても敵は弱いまま。序盤の資金稼ぎ・とくぎのレベル上げは 入口寄りで、装備とパーティが整ってから奥へ踏み込むのが安全です。
05
ブラックサンタ戦は闇属性。光属性の技（シャインフラッシュ／ホーリーレイ） なら1.5倍のダメージが狙えます。ホーリーナイトやほしのこ系統を1体編成に入れておきましょう。
06
クリア後は必ずどうくつ最深部でブラックサンタに再戦を。勝つと仲間にでき、 さらに「サンタさま × ブラックサンタ → イヴ」の最終合成ルートが開きます。
